JavaServer Pages (JSP) Basics II

Sang Shin, www.javapassion.com



JSP allows JSP scripting through which you can insert Java code in the JSP pages.  JSP, just like Servlet, supports include and forward.  JSP also lets you display a designated error page.  Just like Servlet, JSP supports various scopes including page scope.  In this lab, you are going exercise these features.
