Servlet Basics II

Sang Shin, www.javapassion.com, class forum



In this hands-on lab, you are going to learn the second part of Servlet basics, which include how to return HTTP status and error code, how to set and get attributes in the various types of scope objects, how to set and retrieve init parameters, how to use include contents from other servlet through Dispatcher, and how to do logging in the Servlet code.

1. GreetingServlet2.java

package servlets;

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;


/**
 * This is a simple example of an HTTP Servlet.  It responds to the GET
 * method of the HTTP protocol.
 */
public class GreetingServlet2 extends HttpServlet {
    public void doGet(HttpServletRequest request, HttpServletResponse response)
        throws ServletException, IOException {
        response.setContentType("text/html");
        response.setBufferSize(8192);

        PrintWriter out = response.getWriter();

        // then write the data of the response
        out.println("<html>" + "<head><title>Hello</title></head>");

        // then write the data of the response
        out.println("<body  bgcolor=\"#ffffff\">" +
            "<img src=\"duke.waving.gif\" alt=\"Duke waving\">" +
            "<h2>Hello, my name is Duke. What's yours?</h2>" +
            "<form method=\"get\" action=\"greeting2\">" +
            "<input type=\"text\" name=\"username\" size=\"25\">" + "<p></p>" +
            "<input type=\"submit\" value=\"Submit\">" +
            "<input type=\"reset\" value=\"Reset\">" + "</form>");

        String username = request.getParameter("username");

        if ((username != null) && (username.length() > 0)) {
            RequestDispatcher dispatcher =
                getServletContext()
                    .getRequestDispatcher("/response");

            if (dispatcher != null) {
                dispatcher.include(request, response);
            }
        }
        else{
            response.sendError(500, "You did not enter your name");
        }

        out.println("</body></html>");
        out.close();
    }

    public String getServletInfo() {
        return "The Hello servlet says hello.";
    }
}

Exercise 2: Scope objects

Learning points:

  • Scope objects are used for saving attrubutes.  These attrubutes can be then accessed within the Servlet code
  • There are various types of scopes and they are ServletContext scope, Session scope, and Request scope.
  • The attributes maintained in the ServletContext scoipe are accessible by any Servlet (and JSPs) in that application
  • The attributes maintained in a Session scope are accessible by Servlets (and JSPs) in that session
  • The attributes maintained in a Request scope are accessible by Servlets (and JSPs) that handles the request
